# Lesson 02: Connecting MCP Servers

Configure real MCP servers and use their tools inside Claude Code.

## What you'll build
- A `settings.json` with the Neon MCP server connected
- A prompt that uses an MCP tool to query a database
- A troubleshooting guide for common MCP connection errors

## Key concepts
- `claude mcp add <name> <command>` adds a server to your config
- `claude mcp list` shows all configured servers and their status
- MCP servers auto-start when Claude Code launches
- Check server status with `/mcp` inside Claude Code
- Common errors: process not found, auth failures, port conflicts
