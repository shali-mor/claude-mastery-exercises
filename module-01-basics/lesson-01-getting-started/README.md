# Lesson 1-1: Getting Started with Claude Code

**Module:** Claude Code Basics
**Estimated time:** 10 minutes
**Course link:** Module 1 → Lesson 1

## What you'll practice

- Installing and verifying Claude Code
- Running your first interactive session
- Using `/init` to generate a CLAUDE.md
- Running a non-interactive one-off prompt with `-p`
