# Slash Command Scenarios — Solutions

**Scenario 1:** Session is heavy, want to keep context but reduce tokens.
```
Command: /compact
```

**Scenario 2:** Review changes before committing.
```
Command: /review
```

**Scenario 3:** Switch to cheapest model for a simple task.
```
Command: /model claude-haiku-4-5-20251001
```

**Scenario 4:** Check session cost.
```
Command: /cost
```

**Scenario 5:** Check active model, permission mode, and CLAUDE.md status.
```
Command: /status
```

**Scenario 6:** Completely fresh start.
```
Command: /clear   (or /reset for a deeper reset)
```

**Scenario 7:** Add a permanent rule for this project.
```
Command: /memory
```
