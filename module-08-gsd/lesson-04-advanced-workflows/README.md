# Lesson 04: Advanced GSD Workflows

Handle mid-phase pivots, multi-milestone projects, and team GSD setups.

## What you'll build
- A mid-phase pivot procedure using `/gsd:insert-phase`
- A multi-milestone roadmap for a growing project
- A team GSD configuration with shared hooks and settings

## Key concepts
- `/gsd:insert-phase` adds an urgent phase between existing phases without losing progress
- `/gsd:pause-work` captures context so you can resume later or hand off to a teammate
- `/gsd:resume-work` restores full context from the pause checkpoint
- Multi-milestone: each milestone is a separate `PROJECT.md` version
- Team GSD: commit `.planning/` to git, use hooks for atomic commits enforcement
