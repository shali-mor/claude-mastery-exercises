# Exercise: Starting & Planning

## Goal
Create a complete GSD project plan for a real feature.

---

## Task 1: Write a GSD-style PLAN.md (20 min)

Open `starter/PLAN.md` — this is what `/gsd:plan-phase` generates.

Fill it in for this phase:
**Project**: Todo API (Node.js + TypeScript + Express)
**Phase 1 goal**: "Implement CRUD endpoints for todos with SQLite storage"

Your PLAN.md must include:
- Phase goal statement
- Task list (numbered, atomic steps)
- Files to create/modify for each task
- Test strategy
- Atomic commit plan (one commit per task)
- Expected duration

---

## Task 2: Research checklist (10 min)

Before planning Phase 1, what would you need to know about the codebase?
List the questions GSD's research agents would answer.
