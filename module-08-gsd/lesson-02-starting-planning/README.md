# Lesson 02: Starting a Project & Planning Phases

Run `/gsd:new-project` and `/gsd:plan-phase` on a real codebase.

## What you'll build
- A PROJECT.md generated by GSD
- A PLAN.md for Phase 1
- A understanding of how GSD researches before planning

## Key concepts
- `/gsd:new-project` spawns parallel researcher agents before writing PROJECT.md
- `/gsd:plan-phase N` creates a detailed PLAN.md for phase N
- The plan includes: task list, file paths, test strategy, atomic commit plan
- GSD verifies the plan covers the phase goal before showing it to you
