# Exercise: GSD Methodology

## Goal
Structure a real project using the GSD framework.

---

## Task 1: Create a PROJECT.md (15 min)

Open `starter/PROJECT.md` and fill in the structure for a fictional project:
**Project**: A SaaS time-tracking app with Slack integration

Complete all sections:
- Project description and success criteria
- Milestone 1 (MVP) with 3 phases
- Each phase with 3-5 tasks
- Definition of Done for each phase

---

## Task 2: GSD vocabulary quiz (5 min)

Open `starter/vocabulary.md` and match each GSD concept to its definition.

---

## Task 3: Command map (5 min)

Complete `starter/command-map.md` — map each GSD slash command to when you use it.
