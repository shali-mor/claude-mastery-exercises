# Lesson 02: Zapier & Make.com

Compare n8n with cloud-hosted automation platforms and know when to use each.

## What you'll build
- A Zapier Zap design calling Claude via webhooks
- A Make.com scenario blueprint for the same workflow
- A comparison matrix to choose the right platform

## Key concepts
- Zapier: managed, easy, 5,000+ integrations, limited customisation
- Make.com: visual, more powerful data transforms, better for complex logic
- n8n: self-hosted, unlimited control, requires infrastructure
- All three can call Claude via HTTP module/action
- Cost model: Zapier/Make charge per task execution; n8n is flat server cost
