# Lesson 04: Choosing the Right Automation Tool

Build a decision framework for automation tool selection.

## What you'll build
- A comprehensive decision matrix
- A "when Claude automation makes sense" guide
- Your personal automation stack recommendation

## Key concepts
- Claude automation ROI = (time saved × frequency) vs (setup cost + API cost)
- High-frequency, low-stakes tasks → Claude + cron/webhook
- Complex multi-step workflows → n8n or Make.com
- Simple trigger-action → Zapier
- Custom code + full control → direct API with Node.js/Python
