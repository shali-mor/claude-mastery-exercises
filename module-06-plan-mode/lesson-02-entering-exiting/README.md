# Lesson 02: Entering and Exiting Plan Mode

Master the mechanics of toggling Plan Mode and the tools available within it.

## What you'll build
- A cheat-sheet for Plan Mode keyboard shortcuts and tools
- A custom hook that auto-activates Plan Mode for risky commands
- Practice sessions entering/exiting Plan Mode

## Key concepts
- `Shift+Tab` toggles Plan Mode on and off
- Inside Plan Mode: `ExitPlanMode` tool signals readiness to execute
- `AskUserQuestion` gets clarification without making changes
- Hook tip: use `UserPromptSubmit` to warn users before destructive operations
- The status bar shows `[PLAN]` indicator when active
