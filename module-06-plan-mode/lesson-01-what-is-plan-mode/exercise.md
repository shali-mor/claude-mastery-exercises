# Exercise: What Is Plan Mode?

## Goal
Develop intuition for when Plan Mode prevents costly mistakes.

---

## Task 1: Scenario analysis (10 min)

Open `starter/scenarios.md`. For each task, decide:
- **Plan first** or **dive in**?
- If plan first: what should the plan cover?
- What's the risk of NOT planning?

---

## Task 2: Plan Mode workflow (10 min)

Open `starter/workflow.md` and fill in the 5-phase Plan Mode workflow:
1. **Activate** — how?
2. **Explore** — what does Claude do in read-only mode?
3. **Clarify** — how does Claude ask questions without making changes?
4. **Present plan** — what should a good plan include?
5. **Execute** — how do you commit to the plan?

---

## Task 3: Anti-patterns (5 min)

Open `starter/antipatterns.md` and describe 3 real mistakes that Plan Mode would have prevented.
