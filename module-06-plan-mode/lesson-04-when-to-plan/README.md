# Lesson 04: When to Use Plan Mode

Build a personal decision framework for Plan Mode.

## What you'll build
- A decision tree for Plan Mode
- A settings hook that enforces planning for high-risk operations
- Your personal Plan Mode policy (for your CLAUDE.md)

## Key concepts
- Rule: if the task is irreversible, touches > 3 files, or has unknown blast radius → plan
- Rule: if the task is exploratory, experimental, or trivially reversible → skip planning
- Team policy: define in CLAUDE.md so everyone uses Plan Mode consistently
- Hooks can enforce policy: block execution without a plan file present
