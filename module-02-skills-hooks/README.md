# Module 02: Skills, Hooks & Commands

This module teaches you how to extend Claude Code with custom automation — skills, hooks, and a deep understanding of memory and context.

## Lessons

1. **Lesson 01** (10 min): Skills vs Hooks vs Commands — distinguish the four extension types (built-in commands, bundled skills, custom skills, hooks) and fix a broken `.claude/` directory.

2. **Lesson 02** (18 min): Writing Skills That Work — build four functional skills: JSDoc generation, security review, commit messages, and code explanation.

3. **Lesson 03** (20 min): Hooks in Practice — implement real-world hooks: content filters, file write auditing, and session notifications.

4. **Lesson 04** (15 min): Team Patterns & CLAUDE.md — structure `.claude/` for team use, write a production-quality CLAUDE.md, and set up shared hooks.

5. **Lesson 05** (12 min): Ready-to-Copy Skills Library — complete four production skills and build a meta-skill that generates new skills.

6. **Lesson 06** (15 min): Memory & Context — set up all four memory layers (global CLAUDE.md, project CLAUDE.md, auto-memory, context window) and practice choosing the right context management command.

## Getting Started

Start at `lesson-01-skills-vs-hooks/` and work through each lesson in order.
