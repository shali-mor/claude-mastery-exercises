# Lesson 04: Team Patterns & CLAUDE.md

Craft a battle-tested `CLAUDE.md` and `.claude/` layout that onboards any teammate in seconds.

## What you'll build
- A complete `CLAUDE.md` covering stack, conventions, and forbidden patterns
- A `.claude/settings.json` with team-safe permission rules
- A shared skills directory structure your whole team can commit and use

## Key concepts
- `CLAUDE.md` is auto-loaded into every Claude Code session — treat it like a senior dev readme
- Sections: stack overview, key files, coding conventions, what NOT to do, CI/CD notes
- Commit `.claude/commands/` to git so every team member gets the same skills
- Use `settings.json` to lock down dangerous operations in shared environments
