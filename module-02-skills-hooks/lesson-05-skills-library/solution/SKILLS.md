# Skills Library

| Command | Trigger | Description |
|---------|---------|-------------|
| `/changelog` | `/changelog 1.2.0` | Generate a CHANGELOG entry from git log since last tag |
| `/debug-plan` | `/debug-plan "TypeError: Cannot read property"` | Structured debugging plan with hypotheses and investigation steps |
| `/refactor-check` | `/refactor-check src/lib/auth.ts` | Safety analysis before refactoring — callers, coverage, complexity |
| `/env-audit` | `/env-audit` | Audit all env vars for documentation, validation, and usage |
| `/new-skill` | `/new-skill "deploy-check: verify deployment health"` | Generate a new skill file from a name and description |
| `/pr-checklist` | `/pr-checklist` | Full PR review: diff summary, test plan, breaking change check |
